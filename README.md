# Call of Roma Client Fixes

Unpacked Chrome extension for small Call of Roma client fixes.

## What It Patches

- fixes tier-4 Rectorate sprites on the world map
- makes server-confirmed wilds open as wilds instead of buildable clearings
- removes stale expired entries from the Arena/Intelligence reminder
- fixes world-map hero selection after switching cities
- adjusts displayed hero troop capacity to match the server limit
- hides the completed-quest popup
- refreshes the completed-quest count after launch
- starts the news bar closed
- removes the outer-page recharge ad/sidebar

No debug logging is enabled.

## Install Unpacked

1. Open Chrome and go to `chrome://extensions`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select the extension folder, meaning the folder that contains `manifest.json`, `content.js`, and `page-patch.js`.
5. Reload the Call of Roma game tab.
